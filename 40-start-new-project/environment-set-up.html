Installation de l'environnement

Installer mon environnement

Pour commencer le projet, il est nécessaire de mettre en place l'environnement de développement.

NodeJS

NodeJS est un framework web serveur permettant d'exécuter du code Javacript sur toute machine en dehors du browser.

La dernière version est téléchargeable ici. Lancer l'installeur Windows, et suivre les instructions, en laissant les valeurs par défault.
